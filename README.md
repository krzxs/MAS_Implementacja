# System Wypożyczalni Samochodów (SWS)
 
Projekt zaliczeniowy MAS - aplikacja webowa do obsługi wypożyczalni pojazdów.
 
**Autor:** Krzysztof Rosłon (s31006)
 
## Stos technologiczny
 
- Java 21, Spring Boot 4.0.6
- Spring MVC + Thymeleaf, Spring Security, Spring Data JPA
- Baza H2 (plikowa), Bootstrap 5
- Build: Gradle (wrapper w repo)
## Uruchomienie
 
```bash
./gradlew bootRun
```
